# sveltekit-ai-bot

A SvelteKit app that provides an API for asking questions about Svelte and SvelteKit. 

## Getting started

```bash
cp .env.example .env # Fill in the .env file with Anthropic API key and other values
nvm i && nvm use
npm i
npm run dev
```

http://localhost:5173

Default login is `admin / admin`

### Wrap up codebase

```
npx ai-digest
```
